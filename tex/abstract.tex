This thesis investigates ROI-aware LiDAR range-image compression as an end-to-end learning problem under constrained bitrate. The implemented pipeline encodes five-channel range images, predicts spatial importance, applies adaptive latent quantization, and reconstructs the signal with a decoder. For publication clarity, experiments are organized into four model families: Uniform Baseline (UB), Adaptive ROI Student (ARS), Adaptive Distilled Student (ADS), and Multi-Scale Distilled Student (MS-ADS).

A core contribution of this draft is an objective-level analysis that links each loss component to an observed failure mode. The documented objective combines reconstruction, rate proxy regularization, weighted importance supervision, optional ROI/background separation, and teacher distillation. Using the fixed February 20, 2026 snapshot, confirmed results are reported for UB/ARS/ADS and preliminary results for MS-ADS. The best confirmed ARS run reached 0.7224 final loss (ResNet), and the best confirmed ADS run reached 1.3821 final loss at \(\lambda_{distill}=0.1\) (ResNet). In the 16-frame native-vs-oracle decomposition, entropy bitrate proxy dropped sharply with oracle ROI control (ARS: 0.8064 to 0.4177; ADS: 0.8687 to 0.4492 in bpp-entropy) while ROI-MSE remained nearly unchanged.

These findings indicate that adaptive quantization itself has substantial bitrate potential, but current performance is constrained by importance-map quality and threshold calibration. Therefore, the immediate research priority is detector-facing calibration and validation rather than further quantizer-form reformulation alone.
